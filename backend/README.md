<p align="center">Backend for Hanabi Take Home Assignment built with NestJS.</p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Enviroment Variables
`
MONGODB_URL : This should have the connection string of the MongoDB database.
`
